# Skill Sum - Math Game

A fun and interactive mental math game built with React, Next.js, TypeScript, and Tailwind CSS. This game helps improve mental math skills through timed challenges with AI-powered features including hints, text-to-speech, and personalized feedback.

## Features

- **Multiple Difficulty Levels**: Easy, Medium, and Hard modes with different number ranges and operators
- **Customizable Timer**: Choose from preset times (30s, 60s, 90s) or set a custom duration
- **AI-Powered Hints**: Get helpful math tricks and tips using Google Gemini AI
- **Text-to-Speech**: Have questions read aloud using Gemini's TTS capabilities
- **Math Jokes**: Enjoy child-friendly math jokes generated by AI
- **Personalized Feedback**: Receive encouraging messages based on your performance
- **Responsive Design**: Works great on desktop and mobile devices
- **Beautiful UI**: Colorful, game-like interface with smooth animations

## Tech Stack

- **Frontend**: React, Next.js 15, TypeScript, Tailwind CSS
- **AI Integration**: Google Gemini API for text generation and text-to-speech
- **Styling**: Custom CSS with Poppins font and game-themed colors
- **Animations**: CSS keyframes for shake effects and bounce-in animations

## Getting Started

### Prerequisites

- Node.js 18+ and npm
- Google Gemini API key

### Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   cd skill-sum-nextjs
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Set up environment variables:
   ```bash
   cp .env.example .env.local
   ```
   
   Add your Gemini API key to `.env.local`:
   ```
   GEMINI_API_KEY=your_gemini_api_key_here
   ```

4. Run the development server:
   ```bash
   npm run dev
   ```

5. Open [http://localhost:3000](http://localhost:3000) to play the game!

## Game Rules

1. **Choose Settings**: Select your preferred game duration and difficulty level
2. **Solve Problems**: Answer math questions as quickly and accurately as possible
3. **Use Features**: 
   - Click the speaker button (🔊) to hear questions aloud
   - Click "✨ Hint" for AI-generated math tips
   - Try the math joke feature for some fun!
4. **Beat the Clock**: Get as many correct answers as possible before time runs out
5. **View Results**: See your score and receive personalized feedback

## Difficulty Levels

- **Easy**: Addition and subtraction with numbers 1-15
- **Medium**: Addition and subtraction with numbers 1-30
- **Hard**: All four operations (+, -, ×, ÷) with numbers 1-50

## API Routes

- `/api/gemini-text`: Generates hints, jokes, and feedback using Gemini
- `/api/gemini-tts`: Converts math questions to speech using Gemini TTS

## Components

- `SkillSumGame`: Main game component managing state and flow
- `StartScreen`: Game setup and configuration
- `GameScreen`: Active gameplay with questions and answers
- `EndScreen`: Results display with AI-generated feedback
- `JokeModal`: Modal displaying math jokes
