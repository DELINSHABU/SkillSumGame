# Skill Sum - Mental Math Game

A fun and interactive mental math game built with React, Next.js, TypeScript, and Tailwind CSS.

---

## Project Overview

**Skill Sum** is an educational math game designed to improve mental math skills through timed challenges. Players solve math problems while racing against the clock, with AI-powered features like hints, text-to-speech, and personalized feedback.

---

## Core Features

### 1. Quick Play Mode
- **Difficulty Levels**: Easy (1-15), Medium (1-30), Hard (1-50 with all operators)
- **Timer Options**: 30s, 60s, 90s, or custom duration
- **Operators**: Addition/subtraction (Easy/Medium), all four operations (Hard)

### 2. Campaign Mode (50 Levels)
A progressive learning system organized into 8 worlds:
- **World 1**: Addition Foundation (7 levels)
- **World 2**: Subtraction Strategies (5 levels)
- **World 3**: Multiplication Basics (13 levels)
- **World 4**: Division Tricks (10 levels)
- **World 5**: Advanced Addition/Subtraction (4 levels)
- **World 6**: Mixed Operations (3 levels)
- **World 7**: Larger Numbers (4 levels)
- **World 8**: Mental Math Mastery (3 levels)
- **Final**: The Grand Master challenge

### 3. AI-Powered Features
- **Math Hints**: Get helpful tips using Google Gemini AI
- **Text-to-Speech**: Questions read aloud
- **Math Jokes**: Child-friendly jokes generated by AI
- **Personalized Feedback**: Performance-based encouraging messages

### 4. Leaderboard System
- Local storage-based score tracking
- Top 100 scores saved
- Filter by difficulty and time
- JSON-based backend persistence

### 5. Achievements System
- First Step: Answer first question correctly
- Speed Demon: 5 correct in a row
- Perfect!: Complete with no wrong answers
- Math Wizard: 20+ on hard mode
- Streak Master: 10 answer streak

## How the Game Works

### Game Flow
1. **Start Screen**: Choose Quick Play or Campaign
2. **Configuration**: Select difficulty, timer, enter username
3. **Gameplay**: Answer math questions displayed one at a time
4. **Scoring**: +1 point per correct answer
5. **End Screen**: View score, achievements, personalized feedback

### Question Generation
Questions are generated based on skill type:
- **makeTen**: Pairs that sum to 10 (e.g., 3+7)
- **addTen**: Add 10 to any number (e.g., 23+10)
- **addNine**: Add 9 using the trick (add 10, subtract 1)
- **doubles**: Same number added (e.g., 7+7)
- **nearDoubles**: Consecutive numbers (e.g., 7+8)
- **multiplyByTen**: Multiply by 10 (add zero)
- **multiplyByFive**: Multiply by 5 (x10, then half)
- **multiplyByEleven**: Special 2-digit trick
- **integerDivision**: Always whole number results

### Scoring
- Base score: 1 point per correct answer
- Bonus: +0.5 × max streak at end
- Streak tracking for achievements

### API Endpoints
- `GET /api/leaderboard`: Fetch top scores (supports filtering)
- `POST /api/leaderboard`: Add new score

### State Management
- React useState for game state
- localStorage for persistence:
  - `skillsum_progress`: Campaign level progress
  - `skillsum_leaderboard`: Top scores
  - `skillsum_playername`: Saved username
